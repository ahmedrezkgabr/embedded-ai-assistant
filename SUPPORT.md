# Support

## Getting help

### GitHub Discussions

For questions, ideas, and general discussion:
https://github.com/ahmedrezkgabr/embedded-ai-assistant/discussions

### GitHub Issues

For confirmed bugs and feature requests:
https://github.com/ahmedrezkgabr/embedded-ai-assistant/issues

Use the issue templates — they help us help you faster.

### Before asking

Run these commands and include the output:

```bash
bash scripts/check.sh
bash scripts/test.sh
```

Check the troubleshooting sections in:

- README.md
- docs/voice-pipeline.md (for voice issues)
- yocto/README.md (for build issues)
- llm/README.md (for LLM issues)

## What is supported

- Bug reports with reproduction steps
- Questions about the codebase
- Hardware compatibility questions (RPi5, QEMU)
- Yocto build issues

## What is not supported

- General Yocto/BitBake tutorials
- General LLM or AI questions unrelated to this project
- Hardware not listed in docs/hardware.md

## Response time

This is a personal open-source project.
Best-effort response, typically within a few days.
